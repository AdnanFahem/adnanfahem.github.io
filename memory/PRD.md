# B21 Security Ltd - Website PRD

## Original Problem Statement
Create a modern, professional, and fully responsive website for a UK-based security services company that provides trained security guards for businesses, events, construction sites, and residential areas.

## What's Been Implemented (April 2026)

### Pages
1. **Home Page** - Hero section with CTA, services overview, industries section, testimonials, stats
2. **About Us** - Company intro, mission/vision, values, timeline, why choose us
3. **Services** - All 7 services with detailed pages:
   - Manned Guarding
   - Event Security
   - Construction Site Security
   - CCTV Monitoring
   - Mobile Patrols
   - Retail Security
   - Dog Handling / K9 Security (NEW)
4. **Industries** - Corporate, Retail, Construction, Residential, Events
5. **Pricing/Get Quote** - Pricing packages (Basic, Standard, Premium) + quote form
6. **Careers** - Job application with CV upload capability
7. **Contact** - Contact form, FAQ accordion, Google Maps embed

### Features
- Fully responsive design (mobile, tablet, desktop)
- Dark theme with gold accents (professional security aesthetic)
- Cookie consent banner (GDPR compliance)
- Working contact/quote/career forms (MongoDB storage)
- FAQ accordion
- Trust badges marquee
- Smooth animations (Framer Motion)
- SIA Approved Contractor branding

### Tech Stack
- Frontend: React 19, Tailwind CSS, Framer Motion, Phosphor Icons
- Backend: FastAPI, Motor (MongoDB async)
- Database: MongoDB
- UI Components: Shadcn/UI
